What GenomeHub Does

The network, not the cloud.

01 / Distributed Seeding

Any node can host any assembly.

Coverage grows with the network. No central servers. A genome seeded by one node is available to all — as long as at least one peer holds it.

● Content-defined chunksBLAKE3 verified
● Content-addresseddedup guaranteed
● Delta + segmentsdownload only diffs
02 / Verified Integrity

Every segment cryptographically signed.

Reference assemblies are too critical to trust without verification. Every segment is re-hashed on receipt and the manifest signature is checked against the origin's key. Corruption, tampering, and untrusted relays are refused automatically.

● ed25519 manifestorigin-signed
● BLAKE3 segmentsre-hashed on arrival
● Verified via peerstrust survives relay
03 / Open & Noncommercial

Source-available. Free for science.

Built for researchers, not profit. Free to use, fork, and run for any noncommercial purpose — academic, government, and nonprofit use included. The reference implementation is a single Go binary with a curl-friendly HTTP API.

● PolyForm NC 1.0noncommercial
● HTTP read APIcurl-friendly
● Written in Gosingle binary
How It Works

Four steps to seeding.

01
▶
Import

Bring a genome from a local FASTA. The node chunks it into content-defined segments automatically.

$ genomehub import --fasta genome.fa
02
▶
Segment

Sequence is split into content-defined chunks, each indexed by BLAKE3. The manifest is ed25519-signed and published to the origin.

✓ content-addressed · deduped
03
▶
Serve

Your node responds to peer requests over HTTP/2. Segments are served directly; the tracker is only consulted for discovery.

GET /segments/{blake3-hash}
04
Sync

The tracker coordinates availability. Nodes discover peers, fetch missing segments, and propagate delta updates automatically.

↑ delta synced · 3 peers

Get Started

Run a node in minutes.

A single binary, no dependencies. Bring your own storage — a few terabytes is enough to seed the most common reference assemblies.

Terminal bash
# Install (macOS / Linux)
$ curl -fsSL https://raw.githubusercontent.com/luizeduardocarvalho/genomehub/main/install.sh | sh
# Download a genome (verified)
$ genomehub download --assembly Phypa_V3 --output ppatens.fa
✓ manifest Phypa_V3 signature verified
▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ downloaded · all chromosomes OK
# Become a seeder (laptop: auto public tunnel)
$ genomehub seed --tunnel
